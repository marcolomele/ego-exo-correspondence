Dataset resembling the health scenario dataset from EgoExo. Produced by us manually at Gio's house (hence the name) with the purpose of testing the robustness of O-MaMa. 

Download from [Google Drive folder](https://drive.google.com/drive/folders/1a9mk2fO4zTvbk7cbunHxcsdc559jAZ7J?usp=drive_link) (requires access confirmation for privacy).

Use this folder for a small out-of-dataset inference as well as to understand the structure of the `root` folder.

Note: `precompued_features` folder missing since inference for this dataset is run with features precompuetd on-the-fly using `../../src/O-MaMa/main_eval.py`.
